# WPF-Keypass
Password Manager

App for stroring passwords in the database safely.

<h3>Features</h3>

<ul>  
  <li>Import/Export File</li>
  <li>Backup to file</li>  
  <li>Friendly design</li>  
</ul>

<h3>Technologies</h3>

<ul>
  <li>Entityframework</li>
  <li>SQLite</li>
  <li>Cryptography Synchronous</li>
  <li>UnitOfWork Design Pattern</li>
  <li>Linq json, XML</li>
</ul>
